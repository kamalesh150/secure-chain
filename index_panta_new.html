ℹ️ How It Works

1️⃣ Select your file — a SHA-256 hash is computed in your browser.


2️⃣ File hash + name are stored permanently on the blockchain.


3️⃣ IPFS hash (CID) references the actual file content off-chain.


4️⃣ Grant / Revoke access to any wallet address at any time.


5️⃣ Anyone can verify a certificate using the document ID + IPFS hash.
